---
title: "Retrieve an airtime order"
description: "Check the current status of an airtime order."
---

**GET** `/v1/airtime/{id}`

<Info>
  For real-time updates, prefer [webhooks](/webhooks) instead of polling.
</Info>

<RequestExample>
```bash cURL
curl -X GET https://api.tupay.africa/v1/airtime/ABCDEFG250101 \
  -H 'Authorization: Bearer {token}'
```
</RequestExample>

### Path parameter

| Parameter | Description |
|-----------|-------------|
| `id` | Transaction ID or your internal reference |

---

<ResponseExample>
```json Response
{
  "id":        "ABCDEFG250101",
  "reference": "ref1234",
  "status":    20,
  "message":   "Success"
}
```
</ResponseExample>

<ResponseField name="id" type="string">
  Tupay transaction ID.
</ResponseField>

<ResponseField name="reference" type="string">
  Your internal reference.
</ResponseField>

<ResponseField name="status" type="number">
  See [Status Codes](/status).
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable status.
</ResponseField>
